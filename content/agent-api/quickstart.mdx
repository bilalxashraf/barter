# Quickstart

## Install + run

```bash
cd agent-api
cp .env.example .env
npm install
npm run dev
```

Server runs on `https://api.barterpayments.xyz`.

---

## Required environment variables

```bash
PORT=4010
OPENAI_API_KEY=your-key
REQUIRE_API_KEY=false
API_KEY=your-secret
REQUIRE_AGENT_KEY=true

BASE_RPC_URL=...
ETHEREUM_RPC_URL=...
OPTIMISM_RPC_URL=...
ARBITRUM_RPC_URL=...
BSC_RPC_URL=...

CDP_API_KEY_ID=...
CDP_API_KEY_SECRET=...
CDP_WALLET_SECRET=...
CDP_NETWORK_ID=base
```

---

## Health check

```bash
curl -s https://api.barterpayments.xyz/health | jq
```

---

## First autonomous action

### 1) Create an agent API key
```bash
curl -s https://api.barterpayments.xyz/agent/keys/create \
  -H "Content-Type: application/json" \
  -d '{"agentId":"demo-agent"}' | jq
```

### 2) Execute
```bash
curl -s https://api.barterpayments.xyz/agent/execute \
  -H "Content-Type: application/json" \
  -H "X-API-Key: ak_..." \
  -d '{"agentId":"demo-agent","prompt":"Swap 50% of my ETH to USDC on Base","chain":"base","slippageBps":50}' | jq
```
