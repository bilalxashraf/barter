# API Reference

## Core

- `GET /health`
- `GET /chains`

## Intent

- `POST /intents/parse`
- `POST /intents/execute`

## Tokens

- `POST /tokens/resolve`

## Quotes

- `POST /quotes/swap`
- `POST /quotes/bridge`

## Wallets

- `POST /wallet/analyze`
- `POST /wallet/balance`

## Transactions

- `POST /tx/broadcast`
- `GET /tx/:hash`

## Orders

- `POST /orders/limit/build`
- `POST /orders/limit/submit`
- `GET /orders/limit/:id`
- `POST /orders/limit/:id/cancel`

## Agent

- `POST /agent/keys/create`
- `GET /agent/keys/:agentId`
- `POST /agent/wallets/create`
- `GET /agent/wallets/:agentId`
- `GET /agent/wallets/:agentId/list`
- `POST /agent/execute`
